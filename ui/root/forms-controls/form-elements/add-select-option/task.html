在 select 元素中添加一个选项

下面是一个 <select> 元素：

<select id="genres">
  <option value="rock">Rock</option>
  <option value="blues" selected>Blues</option>
</select>

使用 JavaScript 来实现：

  1. 显示所选选项的值和文本。
  2. 添加一个选项：<option value="classic">Classic</option>。
  3. 使之变为可选的。

请注意，如果你已正确完成所有事项，那么 alert 应该显示 blues。
